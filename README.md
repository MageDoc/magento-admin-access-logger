# magento-admin-access-logger
Log all admin actions for Magento 1, relays on "controller_action_predispatch" event.

Request Logged Data:
- user_id.
- username
- controller_module.
- controller_name.
- action_name.
- parameters.

Comes with a configurable cronjob cleaner.
